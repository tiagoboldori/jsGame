# JsGame

Jogo simples de zumbi feito usando Javascript.
Aplicado conceitos de design patterns de forma simples para fins de estudo.





# Como rodar:
- Basta baixar o codigo mantendo a estrutura de pastas e abrir o index.html utilizando o navegador de sua preferência



# Funcionalidades implementadas:
- Movimentação de jogador
- Movimentação de inimigos
- Ação de jogador (atirar)
- Sistema de ondas (incluindo sistema de dificuldade dinâmico)
- Sistema simples de colisão
